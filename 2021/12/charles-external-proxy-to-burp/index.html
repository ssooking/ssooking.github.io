Browser-> Charles -> Burp

将Charles/Proxyman等作为中间代理，把抓取的数据包传递给burpsuite。

首先

  • Charles监听：127.0.0.1:8888

  • BurpSuite监听：127.0.0.1:8080

1、设置浏览器（或者APP）代理端口为Charles

2、Charles开启代理转发到Burp上

Proxy—-External Proxy Settings，都配置上Burp的本地监听端口。

Web Proxy和Secure Web Proxy都配置为127.0.0.1:8080
